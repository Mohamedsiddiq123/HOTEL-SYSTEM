# 🏨 Hotel Management System

---

## 📌 Project Description
This project is a Hotel Management System developed using Oracle SQL and PL/SQL.  
It is designed to manage hotel operations such as customers, rooms, reservations, payments, and services.

## 🗄️ Database Structure

The system contains the following tables:

- Customers  
- Rooms  
- Employees  
- Reservations  
- Payments  
- Services  

---

## ⚙️ PL/SQL Components

This project includes:

### 1. Tables
Used to create the database structure.

### 2. Inserts
Used to insert sample data into tables.

### 3. Procedures
Used to perform operations and display customer information.

### 4. Functions
Used to return calculated values such as:
- Total payments per customer
- Number of reservations

### 5. Anonymous Blocks
Used for testing and simple PL/SQL execution.

### 6. Window Functions
Used for data analysis:
- ROW_NUMBER
- RANK
- DENSE_RANK
